Use network throttling to simulate a slow network connection.

  1. Open the Chrome DevTools
  2. Navigate to the Network tab
  3. Click the gear icon and select "Throttling"
  4. Select "Slow 3G" from the dropdown
  5. Select the Policy Type and Timeout Interval
  6. Click the "Subscribe" button
  7. Observe the renegotiation attempts and the status of the subscription
  8. Reset the network throttling to unlimited after a few reconnection attempts
